# contentAPI


## 安裝

- 安裝GO (https://go.dev/dl/)
- 完成安裝後 在根目錄執行 go build 建立exe
- 執行exe 

## API

URL localhost:8099/add

| 參數欄位      |參數key        | 型態  |
| -------------|:-------------:| -----:|
| 姓名          | name         | string |
| 內容          | content      | string(50) |
| 手機          | phone        | string(10) |
| 信箱          | email        | string |

data.json 存放輸入資料
